Add a customizable 'goto top' button to your WordPress website.

== Description ==

The WPSniper Goto Top plugin adds a customizable 'goto top' button to your WordPress website, allowing users to quickly navigate back to the top of the page with a single click. With customizable button text, icon, and style options, you can easily match the button to your site's design and branding.

Features:
- Add a 'goto top' button to your WordPress site
- No setup needed. Just install and activate the plugin to get started
- Unlimited Color Options
- Customize the button color, border color, border radius, border style to match your site's design and branding
- Customize position of the button
- Compatible with all modern web browsers
- Lightweight and easy to use

== Installation ==

1. Log in to your WordPress admin panel and go to Plugins -> Add New
2. Type WPSniper Goto Top in the search box and click on search button.
3. Find WPSniper Goto Top plugin.
4. Then click on Install Now after that activate the plugin.
5. Go to Theme Customization option and click on WPSniper Goto Top to customize the button text, icon, and style.

== Frequently Asked Questions ==

Q: How do I customize the 'goto top' button?
A: Go to Theme Customization option and click on WPSniper Goto Top to customize the button text, icon, and style.

Q: Is the plugin compatible with all WordPress themes?
A: The plugin should work with most WordPress themes. However, some themes may require additional customization to ensure proper compatibility.

Q: Can I change the position of the button?
A: Yes, you can choose where the button appears on your page by changing the "Offset from bottom" option in the plugin settings.
